# The file in this branch, ETFs_Risk_Monitoring_Dashboard.py, is to build the project’s raw input data. It pulls historical market data for 12 ETFs, extracts adjusted close prices, computes daily returns, and exports clean files that can be used for portfolio construction, risk metric calculation, and dashboard updates.

# The script downloads data for the following 12 ETFs: DBC, EEM, EFA, GLD, HYG, IEF, IWM, LQD, QQQ, SPY, TLT, and VNQ. The time frame that I use for this project is from 03/02/2021 to 02/27/2026. The script is written to reflect real-time data instead of a fixed interval from 03/02/2021 to 02/27/2026. In other words, this script pulls data in a 5-year interval as of the day that users run it. 

These ETFs were chosen to represent multiple major asset classes: 
1. DBC: broad commodities
2. EEM: emerging market equities
3. EFA: developed international equities
4. GLD: gold
5. HYG: high-yield corporate bonds
6. IEF: intermediate-term U.S. Treasuries
7. IWM: U.S. small-cap equities
8. LQD: investment-grade corporate bonds
9. QQQ: U.S. large-cap growth equities / Nasdaq-100
10. SPY: broad U.S. large-cap equities / S&P 500
11. TLT: long-term U.S. Treasuries
12. VNQ: U.S. real estate / REITs

The script performs the following steps: 
1. Downloads historical ETF data: Historical daily price data is pulled using yfinance.

2. Extracts adjusted close prices because they better reflect return-relevant price history after splits and distributions.

3. Builds a clean price table: All ETF price series are aligned by date and combined into one table.

4. Computes daily simple returns

5. Exports reusable datasets

Expected outputs generated by this script include:

1. prices.csv
2. returns.csv

Main dependencies:
1. pandas — data manipulation
2. numpy — numerical operations
3. yfinance — market data download
4. openpyxl — Excel file support
